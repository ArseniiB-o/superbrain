## Accumulated Knowledge
- Security first: always start OWASP Top 10 checklist
- Authentication: most common critical flaw in web apps
- Never approve deploy with CRITICAL or HIGH unresolved security issues
- Code quality: cyclomatic complexity > 10 is a red flag
- Always check for hardcoded secrets (grep for api_key, password, secret)
