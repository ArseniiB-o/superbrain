# Team Memory: frontend
Last updated: 2026-03-26

## Accumulated Knowledge
- Prefer React with TypeScript for all new projects
- Always use functional components + hooks, never class components
- Accessibility: WCAG 2.1 AA minimum — always add aria-labels
- Performance: check Core Web Vitals (LCP < 2.5s, FID < 100ms, CLS < 0.1)
- State management: Zustand for simple, Redux Toolkit for complex
- Styling: Tailwind CSS preferred; CSS Modules for component isolation
